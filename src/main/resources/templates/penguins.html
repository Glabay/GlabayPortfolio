🐧 Project Penguins 🐧

I am aiming for a retro Mario style, 2D, side-scrolling platformer. There will be enemies and challenges awaiting you while you learn about different types of Penguins and how they live and struggle each and every day.

The idea so far is the player will be at a main page when first launching the game, and you are offered a few options: “Play”, “Settings”, “Credits”, and “Quit”.

Options:

  • Play: You are then taken to a screen where up to 3-4 save slots will be there for you to choose a game file from that you want to play, with options to remove a saved file.
  • Settings: Here you can change things like the volume, sound effects, test server connection, and other things I haven’t thought of yet, like key binds?
  • Credits: This will give credit where it is due, not much else can be said.
  • Quit: This will exit the game, but come on… why would you ever click that option.
